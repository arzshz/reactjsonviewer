# React JSON Viewer
This is a simple ReactJS application that allows you to paste JSON data into a textarea. If the JSON is valid, it displays the structured JSON data in a colorful format similar to how Firefox or online JSON viewers present JSON files. If the JSON is invalid, it shows an error message.

# Getting Started
Follow these instructions to get a copy of the project up and running on your local machine for development and testing purposes.

# Prerequisites
Make sure you have Node.js installed on your machine. If it didn't installed you can install that using(Debian base OS):
curl -fsSL https://deb.nodesource.com/setup_16.x | sudo -E bash -
sudo apt-get install -y nodejs

# Installing
Enter these in terminal:
1. npx create-react-app json-viewer
2. cd json-viewer
3. npm install react-json-view
Clone the repository in another directory and copy and replace that into json-viewer/src/

#Running
Enter this in terminal: npm start
